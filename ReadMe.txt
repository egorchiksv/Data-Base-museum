База данных была создана после посещения Зоологического музея Московского университета.
База данных включает в себя таблицу название музея, таблица организаций проводивших ремонт музея, таблица сотрудников, таблица экскурсий, таблица выставочных залов, таблица этажей, таблица экспанатов, таблица авторов экспанатов и таблица организаций реставраторов экспанатов.
В базе данных указаны связи таблиц между собой.
